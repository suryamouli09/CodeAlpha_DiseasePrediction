# Diabetes Prediction System

## Project Overview
This machine learning project predicts whether a patient is diabetic or not using medical data.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Machine Learning Algorithm
- Random Forest Classifier

## Features
- Predict diabetes using patient data
- Accuracy evaluation
- Confusion Matrix
- ROC Curve
- Modern web interface

## Dataset
Pima Indians Diabetes Dataset

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Train model

py train_model.py

3. Run Flask app

py app.py

## Output
The system predicts:
- Diabetes Detected
OR
- No Diabetes
